Smart Contracts exercises

Nel file ERC20.sol ho creato un token ERC20 chiamato HW1_Token (HW1) deployato su testnet Amoy di Polygon. Nello smart contract ho implemenatto le funzioni di mint e burn.
Lo smart contract è visualizzabile su polygonscan al seguente link: https://amoy.polygonscan.com/token/0xa10B69b0185305a747F2e4D843F540A3a7507107

Nel file ERC721.sol ho creato un NFT chiamato HW2_ERC721 (HW2) e deployato su testnet Amoy di Polygon.
Il link per vedere lo Smart Contract è il seguente: https://amoy.polygonscan.com/token/0x8C16bb8f9605A886C6a87cE3b56574FE3beCA507

Nel file ERC1155.sol ho creato uno smart contract con 5 assets utilizzabili in un ipotetico gioco di sopravvivenza, come un arco, delle frecce, un elmetto, delle mele, e delle bottiglie d'acqua.
Lo Smart Contract è visualizzabile al seguente link: https://amoy.polygonscan.com/token/0xB0984933cB730dacC754594c6AcC43d71E6E47A2
La visualizzazione degli NFTs è ancora in fase di sviluppo, perché su polygonscan non viene fornita, mentre OpenSea ha disabilitato la relativa funzione per le testnet.
